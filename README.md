Initial and final structures of nanoparticles in water, membranes in water, and nanoparticles interacting with membranes in water

- NP18 -> Guanidinium
- NP19 -> Dimethyl amine
- NP20 -> Monomethyl amine
- NP21 -> Ammonium

- POPC2 -> Pure POPC bilayer
- POPG2 -> POPC:POPG (9:1) bilayer
